# notes-again

third and, hopefully, final note app design

# features

## notes:

    * pinned
    * archived
    * tags
    * favorite (tag?)
    * checkbox
    * trash (delete after alloted time has passed)

## individual note structure:

    * title
    * body
    * tags (custom categories)
    * id (unique id)
    * checkbox (boolean)

## array storage for notes:

    * default / active
    * archived
    * pinned
    * tagged / favorite?
    * reminders?
    * trash

## possible features:

    * text editor / text formatting
    * import images
    * links
    * reminders
    * drawings
    * audio recordings
    * undo / redo
